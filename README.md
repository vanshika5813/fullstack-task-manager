# Full Stack Task Manager

A full-stack task management application built using the MERN stack.

## Features

- User Registration
- User Login
- JWT Authentication
- Protected Routes
- Add Tasks
- Delete Tasks
- Mark Tasks as Complete
- Search Tasks
- Responsive UI

## Tech Stack

### Frontend
- React
- Vite
- Axios
- React Router

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT
